Usando estilo externo

O que é isso?

Basicamente o estilo externo é aqule que fica em outro arquivo/página, isso facilita na hora de definir um estilo para varias páginas ao mesmo tempo

Para usar esse estilo basta criar um novo arquivo que termine em (.Css)

Usando o CSS externo

Após criar o arquivo das CSS, basta criar o estilo que tu queres.

Para coocar no seu site, basta colocar a tag LINK REL='stylesheet' HREF='endereço de sua pagina de estilo'

Após colocar isso, o estilo desse arquivo será aplicado em todas as páginas que esse link foi inserido
