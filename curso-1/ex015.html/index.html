Usando áudio no site

Para colocar um áudio em seu site é necessario usar a tag AUDIO

Para que o áudio funcione é necessario colocar a tag CONTROLS dentro da tag AUDIO

Podemos também usar outras formas

Abrindo a tag AUDIO e colocando a tag SOURCE

Exemplo:

        
            <audio preload='(auto,metadata,nome)' controls loop>

                <source src='link.mp3' type='audio/seu formato (mp3)'> 

            </audio>
