Aprendendo mais formatações de textos

Marcador

Para marcar algo no texto basta usar a tag Mark, assim, vai poder marcar o texto, também podemos mudar a cor do marcador usando as CSS3!

Textos pequenos / grandes

Podemos deixar Algo grande usando as CSS3, também podemos usar a tag BIG mas ela pode parar de funcionar!

Podemos deixar Algo pequeno usando a tag SMALL, assim, deixando o que quiser em pequeno, também podemos usar as CSS3

Textos apagados

Podemos usar a tag DEL para corrigir algo no texto!

Texto inserido

Podemos inserir algo a(Traço abaixo das letras) em nosso texto usando a tag INS

Podemos também usar a Tag U para inserir algo no texto (não semantica)

Texto Sobescrito

Podemos usar a tag SUP dessa forma: x20 + 30, ao invés de colocar x20 + 30

Texto Subescrito

Podemos usar a tag SUB dessa forma: H2O, ao invés de colocar H2O
